# pc_sales_project
I built a data warehouse to analyze PC sales performance using SQL Server. 

The project involved designing dimension tables (dim_customer, dim_date, dim_channel,dim_priority,dim_location,dim_employee,dim_payment_method,dim_pc,dim_shop)  and a fact table (dim_fact_sales) to track sales metrics like revenue, profit, and units sold.

Key Components:Extracted data from csv file, transformed it, and loaded it into the warehouse using stored procedures.

Data Modeling: Star schema design for efficient querying and reporting.


Analysis: Created dashboards (PowerBI/Excel) to show sales trends, top customers, and channel performance.

Tools Used:SQL Server for data modeling and ETL.

Goal:The warehouse enables stakeholders to track PC sales performance, identify growth opportunities, and make data-driven decisions.
